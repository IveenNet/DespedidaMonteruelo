Descargar docker-compose.yml Descargar .env.example
2) Importa workflows en n8n

Workflows gemelos (DVWA / Juice Shop). Tras importar, sustituye REPLACE_WITH_* por secretos/vars reales.

DVWA workflow Juice Shop workflow
3) Instala OpenClaw + gateway

Instala desde openclaw.ai, haz onboard y deja el gateway persistente con systemd --user.

Los comandos y el perímetro (allowlist) están detallados en la sección OpenClaw de estas slides.
4) Verifica “output”

La salida buena es: resumen corto para móvil (Telegram) + trazabilidad en repo (informes/PR). Si falta contexto, revisa Workspace y allowlist.
